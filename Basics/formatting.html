HTML Formatting Elements

Formatting elements were designed to display special types of texts:

  • < b >

    • - This text is bold

  • < strong >

    • - This text is important!

  • < i >

    • - This text is italic

  • < em >

    • - This text is emphasized

  • < mark >

    • - This is marked text

  • < small >

    • - This is some smaller text.

  • < del >

    • - deleted text

  • < ins >

    • - inserted text

  • < sub >

    • - H2O is water

  • < sup >

    • - 22 = 4
